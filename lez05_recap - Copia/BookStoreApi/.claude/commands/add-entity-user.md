# Minimal API CRUD Skill

Create a new Model, Service and Repository with CRUD endpoints inside `@Program.cs`.

## Rules:
- Keep everything beginner-friendly
- Use an in-memory list as database
- Do not use Entity Framework
- use repositories or services
- Add:
  - GET all
  - GET by id
  - POST
  - PUT
  - DELETE

## Constraints
Before modifying code:
1. Analyze `@Program.cs` and folders: Models, Services and Repositories
2. Propose a short plan
3. Wait for approval unless explicitly asked to implement immediately

## Example usage:
```
Use the minimal-api-crud skill for creating User Model on @Program.cs
```
